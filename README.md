# react-vite-windi-template

已经集成如下：
- vite3
- windi css
- eslint
- prettier
- react-query v4
- vitest (根据情况更新，版本比较低)
- valtio
- react-router-dom
- axios
- husky
- lint-stage

均锁版本，如果需要升级，请自行更新，模板也会不定期更新

## 下载

- 通过`Download ZIP`
- Git clone

```
git clone https://github.com/mrleidesen/react-vite-windi-template.git
```

## 运行

> pnpm install

> pnpm dev
